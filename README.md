# Hikarukimi Web Editor 

一个基于 React + Vite + TypeScript 构建的 Web 代码编辑器。

## 技术栈

- **前端框架**: [React]
- **构建工具**: [Vite]
- **编程语言**: [TypeScript]
- **样式处理**: [Tailwind CSS]
- **图标库**: [Lucide React]
- **编辑器核心**: [@monaco-editor/react]

## 快速开始

### 安装依赖
推荐使用 `pnpm` 或 `npm` 安装依赖：

```bash
pnpm install
# 或者
npm install
```

### 本地开发
启动开发服务器：

```bash
npm run dev
```

### 项目打包
构建生产环境版本：

```bash
npm run build
```
